Features

Eight tabs. One feedback loop.

Every part of the app feeds the next. Today's analytics become tomorrow's hooks. The AI sees exactly what you see, with no hidden context.

Weekly brain reflection

One button summarises the last seven days, names the pillar that's overperforming, and drops three fresh ideas into the bank tagged auto-reflection.

Topic intake from RSS

Pull headlines from Hacker News, dev.to, vendor blogs. Paste a homepage URL and Cadence autodiscovers the feed. One click turns any headline into an angled idea in your voice.

Memory loop

Top performers, voice samples, and discarded ideas are injected into every prompt. Claude learns what worked, what flopped, and what you already rejected. No re-pitching.

Voice-matched drafts

Paste five of your best past posts. Cadence samples three at random for every generation, so the model writes in your rhythm. Not generic AI prose.

Auto-scheduling

Drop ready drafts into the calendar with one click. Spreads them across weekdays at your preferred hours, balanced by pillar percentage.

Analytics import

Drop in LinkedIn's Creator Analytics xlsx. Cadence auto-matches each row to your drafts by body snippet first, date proximity second. No copy-pasting per post.

Engagement helper

Generate three thoughtful comments for any post you paste in. Comments match your voice samples too, so you stay consistent even when commenting on others.

Hook and voice scoring

Score any draft on Hook strength and Voice match, 1 to 10 each, with three concrete improvement notes. Lets you triage which drafts are ready to ship.

Backup & migrate

Move your entire pipeline between machines as a JSON file in 60 seconds. API key is redacted by default. CLI, HTTP, and one-click UI all share the same code path.

What it isn't

Cadence is deliberately not a SaaS.

It runs on your laptop. There is no hosted version, no telemetry, no account, no team plan. The whole app is a single Flask file plus vanilla JavaScript that you read end to end in an afternoon.

What you get

  • Single SQLite file holds every idea, draft, and analytics row
  • Your Anthropic API key, or your Claude Code CLI session, automatically
  • One Python file you can fork, tune, and own forever
  • JSON backup that works between Macs in under a minute
  • MIT licensed, no usage quotas, no rate limits beyond Claude's

What you don't get

  • Hosted SaaS or signup flow. Cadence is local-clone-only
  • LinkedIn API integration. LinkedIn doesn't expose personal post analytics
  • Browser automation or scraping. Against LinkedIn's ToS
  • npm, React, or any build step. Open run.sh and you're done
  • A team mode. Single user, single laptop, by design

60-second install

Three commands, one terminal.

Python 3.10 or later. The setup script creates a virtualenv, installs four dependencies, and starts the server on http://127.0.0.1:5050.

Clone & run

On first boot, an onboarding modal asks for your name, bio, audience, and three of your best past posts to seed the voice library.

No Anthropic API key? Install Claude Code and run claude login once. Cadence auto-detects either backend and falls back transparently. 

git clone https://github.com/nomadicmehul/cadence.git
cd cadence
./run.sh

Roadmap

Shipped, in flight, and on the bench.

Each phase is self-contained. Phase 1 (open-source readiness), Phase 2 (the weekly brain reflection), and Phase 3.b (RSS topic intake with autodiscovery) are live. Phase 3.a (Anthropic web search) and Phase 4 (reusable Claude skills) are next. See the full roadmap for what's in and what's out of scope.

Shipped

Onboarding · pillars · schema migrations · JSON backup · weekly brain reflection · RSS topic intake with autodiscovery · voice-matched comments · memory feedback loop.

Next up

Anthropic web search for topic intake. Semantic memory via Voyage embeddings. Auto-promote winners to the voice library. Hook A/B experiments.

Out of scope

A hosted SaaS. LinkedIn API for personal analytics (doesn't exist). Browser scraping (against ToS). Multi-user mode. npm / build step. See the roadmap for why.
