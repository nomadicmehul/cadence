Backup & migrate

      

        Move to a new computer in 60 seconds. Export downloads a JSON of every
        idea, draft, voice sample, and analytics row. Import wipes the current
        DB and restores it on the new machine. The API key is stripped from
        exports by default — paste it again on the new machine.
      

      

        
        
        
        
      

      

        For headless / scripted use, run
        python app.py export backup.json and
        python app.py import backup.json from the project root.
